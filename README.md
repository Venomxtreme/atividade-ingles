# Technical Guide: Installing and Running []


## Introduction

It is a simple Vue.js and Tailwind CSS application


## Prerequisites

Vue.js, Tailwind CSS, npm/yarn, CSS, HTML 


## Step-by-Step Installation

### Necessary commands

```
git clone https://github.com/Your-profile/your-folder.git
cd your-folder
npm install
npm run dev
```

### 1. Clone the Repository

Choose or create a folder on your desktop or anywhere on your computer, then right-click and select the 'Open in Terminal' option, and then use the command ```git clone https://github.com/Your-profile/your-folder.git``` to clone the desired file. Use the command ```cd your-folder``` to open the folder that was cloned.


### 2. Install Dependencies

Still in the terminal, install the dependencies using the command ```npm install```.



### Start the Development Server

Now, to start the development server, use the command ```npm run dev```  to launch the server. Note: Check if the site displays the correct information; if it does, the site is working properly.
